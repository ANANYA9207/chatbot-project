This project is a rule-based chatbot built as a web application using Flask.
The chatbot responds to user messages using predefined rules and keyword pattern matching without any machine learning model.

It demonstrates how basic Natural Language Processing (NLP) concepts and conversation flow can be implemented using simple conditional logic and integrated into a web interface.
*Features*

- Web-based chat interface

- Keyword-based response system

- Case-insensitive input handling

- Predefined conversational rules

- Instant response using Flask routes (JSON)

- Easy to extend with new rules

**Technologies Used**

- Python

- Flask

- HTML, CSS, JavaScript

- String processing and pattern matching

**How to Run**
1) Install - py -m pip install flask
2) Run -  py app.py
3) Open in browser - http://127.0.0.1:5000
